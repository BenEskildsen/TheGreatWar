    class EntityManager

    The EntityManager is the “database table” of the ecs model. It is responsible for mapping entities (rows) to their components (columns). The manager provides a quick, efficient way to determine which components belong to an entity. It often provides hashes to make it fast to determine which entities have a given component.

    The EntityManager contains all the game state data for the game. Hence, a game can be saved and loaded by marshalling/ unmarshalling the EntityManager.

def get_entities_with_components(*component_classes)
        entity_array = []
        self.each do |entity, comp_bag|
                if component_classes.all? {|comp| comp_bag.has_key?(comp) && comp_bag[comp] != []}
                        entity_array.push(entity)
                end
        end
        return entity_array
end

def has_components(entity, components)
        components.all? { |comp_class|
                self[entity].has_key?(comp_class) and
                self[entity][comp_class].size >= 1
        }
end

    # File app/models/ecs/entity/entity_manager.rb, line 143
def to_s
        string = "EntityManager: {\n"
        self.each { |key, value|
                string += "\t#{key} => #{value}\n"
        }
        string += "\tBoard =>\n"
        @board.each_with_index {|row, index|
                string += "\t\t[Row##{index}\n"
                row.each_with_index {|col, col_index|
                        string += "\t\t\tColumn##{col_index} => #{col}\n"
                }
                string += "\t\t]\n"
        }
        string += "}\n"
end
