    class PieceComponent

    The PieceComponent identifies an entity as an army unit or a piece on the board that a player can control. It also contains a type attribute that identifies which type of unit it is.

    Since there are only a limited number of types of units, the class is structured the same as the TerrainComponent. The initialize function is private while static class instance variables are provided for the different types.
